# AEP Web Tracking Lab

A practice environment for Adobe Experience Platform (AEP) web tracking concepts.

## Live Demo
Access the live application here: [**Live Demo**](https://johnparente97.github.io/aep-web-tracking-lab/site/)
*(Note: Link works after enabling GitHub Pages)*

## Project Structure
- `/site`: The static website pages.
- `/data-layer`: Analytics data layer scripts.
- `/docs`: Documentation and tracking specifications.

## Setup
No build tools required. Just serve the `/site` directory or open the HTML files directly in a browser.
